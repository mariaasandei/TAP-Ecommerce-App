E-Commerce Order System
Course: TAP
Language: C# / .NET 8
Type: Console Application

Overview
A console-based e-commerce order management system built in C#, demonstrating object-oriented design principles including the Strategy Pattern, Interface Segregation, and the Open/Closed Principle.
Features

Product catalogue browsing
Shopping cart management
Order placement and history
Multiple payment methods (Credit Card, PayPal, Cryptocurrency)
Multiple shipping couriers (FedEx, DHL, Local Courier)
Extensible discount campaigns (Percentage, Fixed Amount, Buy X Get Y)
